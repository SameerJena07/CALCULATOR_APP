# 🧮 Calculator App

A simple and responsive **Calculator Web Application** built using **HTML, CSS, and JavaScript**.  
This project performs basic arithmetic operations like addition, subtraction, multiplication, and division.

---

## 🚀 Features

- Perform basic arithmetic operations
  - Addition (+)
  - Subtraction (-)
  - Multiplication (*)
  - Division (/)
- Clear button to reset the calculation
- Responsive calculator UI
- User-friendly interface
- Real-time input display

---

## 🛠️ Technologies Used

- **HTML5** – Structure of the calculator
- **CSS3** – Styling and layout
- **JavaScript (Vanilla JS)** – Calculator logic and functionality

## ⚙️ How to Run the Project

1. Clone the repository

```
git clone https://github.com/SameerJena07/CALCULATOR_APP.git
```

2. Open the project folder

```
cd CALCULATOR_APP
```

3. Open **index.html** in your browser

or simply double click the file.

---

## 💡 How It Works

1. User clicks numbers and operators on the calculator.
2. The input is displayed on the calculator screen.
3. JavaScript processes the expression.
4. The result is displayed when the **= button** is pressed.
